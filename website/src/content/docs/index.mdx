## What is Catalyst?

Catalyst is a **plugin-based toolkit** that extends Claude Code with structured workflows for real-world software development. It provides specialized agents for codebase research, commands for planning and implementation, and integrations with tools like Linear, GitHub, Sentry, and Railway.
