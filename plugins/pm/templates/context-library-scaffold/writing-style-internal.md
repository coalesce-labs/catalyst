# Writing Style Guide (Internal)

## Tone
- Professional but approachable
- Data-driven, cite sources
- Action-oriented (lead with what to do)

## Structure
- Lead with the ask or decision needed
- Use bullet points for lists
- Include TL;DR for documents > 1 page
- Link to source documents, don't copy-paste

## Templates
- Status updates: Context -> Progress -> Blockers -> Next Steps
- Decision docs: Context -> Options -> Recommendation -> Trade-offs
- PRDs: Problem -> Solution -> Metrics -> Timeline
